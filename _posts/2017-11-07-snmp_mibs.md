#### snmp install

```markdown
apt-get update
apt-get install snmp snmp-mibs-downloader
apt-get install snmpd
```

* * *

#### two ways to find snmp directory

```markdown
net-snmp-config --default-mibdirs
```

#### or

```markdown
snmptranslate -Dinit_mib .1.3 2>&1 |grep MIBDIR
```

* * *

#### snmp.conf path

```markdown
/etc/snmp/snmp.conf
```
